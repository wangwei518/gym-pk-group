# gym-pk-group 真实联调 Checklist

这个清单的目标不是讲概念，而是让你在微信开发者工具里一步一步勾着跑通。

使用方式：
- 每完成一项，就打一个勾
- 某一步失败，先不要往后跳，先把当前问题解决
- 推荐按照顺序执行

## K. 建议记录问题的方式

联调时建议单独记一个问题清单，比如：

```markdown
## 联调问题
- [ ] createCircle 后未跳转
- [ ] submitRecord 成功但排行榜没刷新
- [ ] join-circle 页面未正确拿到 inviteCode
```

这样下一轮修问题会非常快。
